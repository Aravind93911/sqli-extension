# Tech Bazzar - SQL Injection Detector Extension

A powerful browser extension for real-time SQL injection detection and protection.

## Features

✨ **Real-Time Detection** - Analyzes form submissions and network requests
🔒 **Automatic Blocking** - Prevents malicious requests from being sent
📊 **Dashboard** - View all detected threats and statistics
🌐 **Global Protection** - Works on all websites
📱 **Easy Setup** - Simple configuration with API key

## Installation

### For Development

1. Clone this repository
2. Navigate to `chrome://extensions/`
3. Enable "Developer mode" (top right)
4. Click "Load unpacked"
5. Select this folder

### For Users

Download from Chrome Web Store (coming soon)

## Setup

1. Click the Tech Bazzar icon in your toolbar
2. Click "⚙️ Setup"
3. Enter your API key from https://dashboard.techbazzar.com
4. Save settings

## Usage

The extension works automatically in the background. When a SQL injection attempt is detected:

1. A warning banner appears
2. The malicious request is blocked (if enabled)
3. The threat is logged to your dashboard

## Configuration

### API Key
Get your free API key from [techbazzar.com](https://techbazzar.com)

### Settings
- **Block Attacks**: Automatically block detected threats
- **Notifications**: Show warning banners
- **Notification Level**: Control alert intensity
